API Reference¶

foxglove¶

This module provides interfaces for logging messages to Foxglove.

See foxglove.schemas and foxglove.channels for working with well-known Foxglove schemas.

class foxglove.Capability¶

A capability that the websocket server advertises to its clients.

class foxglove.MCAPWriter¶

A writer for logging messages to an MCAP file.

Obtain an instance by calling record_file(), or the context-managed new_mcap_file().

If you’re using record_file(), you must maintain a reference to the returned writer until you are done logging. The writer will be closed automatically when it is garbage collected, but you may also MCAPWriter.close() it explicitly.

close()¶

Close the MCAP writer.

You may call this to explicitly close the writer. Note that the writer will be automatically closed for you when it is garbage collected, or when using the context-managed new_mcap_file().

class foxglove.SchemaDefinition(name: str, *, schema_encoding: str, message_encoding: str, schema_data: bytes)¶

A custom schema definition for consumption by Foxglove.

To use Foxglove well-known schemas, use existing Channel and schema definitions.

class foxglove.ServerListener(*args, **kwargs)¶

A mechanism to register callbacks for handling client message events.

on_message_data(client: Client, channel: ClientChannelView, data: bytes) → None¶

Called by the server when a message is received from a client.

Parameters:

  • client – The client (id) that sent the message.

  • channel – The channel (id, topic) that the message was sent on.

  • data – The message data.

class foxglove.StatusLevel¶

The level of a Status message

class foxglove.WebSocketServer¶

A live visualization server. Obtain an instance by calling start_server().

broadcast_time(timestamp_nanos)¶

Publishes the current server timestamp to all clients. If the server has been stopped, this has no effect.

clear_session(session_id=None)¶

Sets a new session ID and notifies all clients, causing them to reset their state. If no session ID is provided, generates a new one based on the current timestamp. If the server has been stopped, this has no effect.

publish_status(message, level, id=None)¶

Send a status message to all clients. If the server has been stopped, this has no effect.

remove_status(status_ids)¶

Remove status messages by id from all clients. If the server has been stopped, this has no effect.

foxglove.new_mcap_file(fname: str) → Iterator[None]¶

Create an MCAP file at the given path for recording.

This is the context-managed equivalent of record_file().

foxglove.record_file(path)¶

Open a new mcap file for recording.

Parameters:

path – The path to the MCAP file. This file will be created and must not already exist.

Returns:

A new MCAPWriter object.

foxglove.start_server(name: str | None = None, host: str | None = '127.0.0.1', port: int | None = 8765, capabilities: List[Capability] | None = None, server_listener: ServerListener | None = None, supported_encodings: List[str] | None = None) → WebSocketServer¶

Start a websocket server for live visualization.

Parameters:

  • name – The name of the server.

  • host – The host to bind to.

  • port – The port to bind to.

  • capabilities – A list of capabilities to advertise to clients.

  • server_listener – A Python object that implements the ServerListener protocol.

  • supported_encodings – A list of encodings to advertise to clients.

foxglove.verbose_off() → None¶

Disable SDK logging.

foxglove.verbose_on(level: int | str = 'debug') → None¶

Enable SDK logging.
